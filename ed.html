1 Euclidean domains and principal ideal domains

Our aim in this section is to understand and axiomatise the properties that a ring must have in order to allow us to carry out divisions and factorisations in an essentially unique way, as we can do in ℤ for instance.

We begin with reviewing some needed notions in ring theory, and fixing the notation that will be used in these notes.

1.1 A brief recap on rings and ideals

A ring is always meant to be a unital ring, that is, a ring is an abelian group (R, +) equipped with a multiplication, denoted by juxtaposition, such that there exists 1 ∈ R with

If the multiplication is commutative, i.e. ab = ba for all a,b ∈ R, then we call R commutative. Most of the rings we will study in this course are commutative rings.

Definition 1.1. Let R be a commutative ring.

A zero divisor in R is an element a ∈ R such that there exists a nonzero element b ∈ R with ab = 0. We say that a is a nontrivial zero divisor if a is a zero divisor and a≠0. (So 0 is always the unique trivial zero divisor in any commutative ring R.)

An integral domain, is a commutative ring R which has no nontrivial zero divisors. For short, an integral domain is written ID.

In other words, if R is an ID, and a,b ∈ R are such that ab = 0, then at least one of a or b must be zero.

For instance, ℤ, ℚ, ℝ, ℂ and ℤ∕2 are IDs, but ℤ × ℤ, ℤ∕4, and more generally ℤ∕n for n not a prime, are not IDs.

Recall that an ideal of a ring R is a subgroup I of (R, +), such that ab,ba ∈ I for any a ∈ R and b ∈ I. These are often called two-sided ideals because I is multiplication-closed on both sides, but since in commutative algebra we consider commutative rings, all ideals are necessarily two-sided, and so we call them simply ideals. In particular, given a ∈ R, then the set aR = Ra = {ar∣r ∈ R} is an ideal. Definition 1.3. Let R be commutative ring. Then R is a principal ideal domain, or PID for short, if R is an integral domain and if any ideal of R is principal.

For instance, in Example 1.2, the first instance gives us all principal ideals, as each of I,J,I + J and I ∩ J are generated by a single element, whereas in the second instance I,J and I ∩J are principal, but I + J isn’t: we need two generators. Indeed, since 2 ∈ I + J, if I + J could be generated by one polynomial, say f ∈ R, then deg f = 0, i.e. f is a (nonzero) integer dividing 2, so f = ±1 or f = ±2. But then, since x ∈ I + J and x± 2R, the generator would be ±1. But ±1 I + J, because this is not a polynomial with even constant term. Therefore I + J is not principal.

Note that the ‘I’ in ID and in PID refer to different concepts. The main instances of PIDs are the ring of integers and the (univariate) polynomial rings over a field. We briefly outline a direct proof of these claims, but as we shall shortly see, these two results are consequences of a ‘stronger’ property that these two rings possess (cf Theorem 1.12).

Proposition 1.4. The ring ℤ of integers is a PID.

Proof.

We already know that ℤ is an ID. So it suffices to show that every ideal I is of the form I = nℤ for some integer n, which we may suppose to be nonnegative (since (-n)ℤ = nℤ). If I = {0}, then I = 0ℤ and we can take n = 0. Suppose now that I≠{0}. We claim that I = nℤ for the smallest positive integer n ∈ I.

The inclusion “I ⊇ nℤ” holds since n ∈ I implies that nk ∈ I for any k ∈ ℤ. Conversely, for the inclusion “I ⊆ nℤ”, let j ∈ I be a positive integer. So j ≥ n, and by the division algorithm, j = nq + r for some nonnegative integers q and r, with 0 ≤ r < n. But then r = j - nq must also be an element of I, by definition of an ideal. By choice of n ∈ I, this forces r = 0, and so j = nq. __

1.2 Euclidean domains and principal ideal domains (ED & PID)

Definition 1.7. Let R be an integral domain. A Euclidean function on R is a function

such that the following two axioms hold:
  • v(ab) ≥ v(a) for all a,b ∈ R \{0};
  • for any a ∈ R and any b ∈ R \{0}, there exist q,r ∈ R such that a = qb + r and either r = 0 or v(r) < v(b).

A Euclidean domain is an integral domain R such that there is a Euclidean function on R. Euclidean domains are often written ED.

The second axiom above is reminiscent of the division algorithm, similarly to what we have seen with the integers and polynomial rings over fields. It is in fact the required property of a commutative ring to define a sensible ‘abstract’ division. Let us take a few examples.

    • Given f,g ∈ F[x] \{0}, we have deg(fg) = deg f + deg g ≥ deg f (recall that since F is an ID, deg fg = deg f + deg g).

  3. Let F be a field, then F is a Euclidean domain for infinitely many Euclidean functions. Indeed, pick n ∈ ℕ0, and let v(a) = n for all nonzero elements a ∈ F. Then,
    • For any a,b ∈ F \{0}, we have v(ab) = n ≥ n = v(a).
    • For any a,b ∈ F with b≠0, then we can write a = b(b-1a)+0, since b-1 ∈ F is defined, and so the second axiom holds with q = b-1a and r = 0.

Definition 1.14. Let R be a (commutative unital) ring, with multiplicative identity 1 and additive identity 0.

  1. An element a ∈ R is invertible if there exists b ∈ R such that ab = 1. We write R× for the set of invertible elements of R, and a-1 for the inverse b of a in R (see Remark 1.17 below).

Remark 1.17.  

  1. Some textbooks refer to an invertible element as a unit in a ring.
  2. Let R be a commutative ring. If an element a ∈ R is invertible with ab = 1, then b is also invertible, and moreover, a can have at most one inverse, since ab = ac = 1 implies that c = c1 = cab = 1b = b. So the notation a-1 for the inverse of a is well-defined.
  3. An invertible element divides any element in R. In particular, u ∈ R×⇐⇒uR = R.
  4. The set R× of invertible elements is a multiplicative group.
  5. The relation ‘a is associated to b in R’ is an equivalence relation.
  6. One could define a ~ b⇐⇒aR = bR.

1.4 Greatest common divisors, Euclidean algorithm and Bézout’s theorem

Notation 1.22. Given elements a,b in a commutative ring R, the notation a∣b means a divides b (in R), that is, there exists c ∈ R such that b = ac.

Definition 1.23. Let R be a commutative ring and a1,…,an ∈ R (n ≥ 2).

  1. An element d ∈ R is a common divisor of a1,…,an if d∣ai for all 1 ≤ i ≤ n.
  2. An element d ∈ R is a greatest common divisor of a1,…,an if d is a common divisor of a1,…,an, such that any common divisor c of a1,…,an is also a divisor of d.

If d is a greatest common divisor of a1,…,an, we write d = gcd(a1,…,an).

Remark 1.24.  

  1. In some textbooks a greatest common divisor is called a highest common factor.
  2. Note that the definition does not guarantee the existence of a greatest common divisor for a given n-tuple.
  3. An invertible element divides any element in a ring. Indeed, if a ∈ R×, then b = 1b = a(a-1b) for any b ∈ R.
  4. The notation gcd(a,b) = d is not really well defined because a greatest common divisor is not unique. But as we shall see below in Proposition 1.26, if it exists, gcd(a,b) is a unique equivalence class of associate elements of R, and so we allow ourselves this slight abuse of terminology.

We can check that the formal definition matches our understanding of a greatest common divisor of integers. For instance, gcd(12, 30) = 6. But we also could have taken -6 instead of 6. Indeed, 6 and -6 have the same integer divisors up to sign.

But as noted above, greatest common divisors need not exist.

The Euclidean algorithm shows that in a Euclidean domain R:

  • given any elements a,b ∈ R (which we may assume to be nonzero and not invertible, to avoid ‘trivialities’), then gcd(a,b) exists, and
  • there exists an algorithm to express a greatest common divisor of a,b as an R-linear combination of a and b.

It is then natural to ask ourselves if there exists a wider class of commutative rings for which these facts hold, but perhaps we do not have an explicit algorithm to calculate greatest common divisors. The answer is given by Bézout’s theorem.

Theorem 1.32 (Bézout’s theorem). Let R be a PID and a1,…,an ∈ R. The following hold.

  1. gcd(a1,…,an) exists in R, and
  2. a greatest common divisor of a1,…,an is an R-linear combination of a1,…,an.

Proof. Let I be the ideal generated by a1,…,an, that is,

Because R is a PID, the ideal I is principal, and so there exists d ∈ R such that I = dR. We claim that d = gcd(a1,…,an):
  • The inclusion ajR ⊆ I = dR implies that aj ∈ dR, or equivalently that d∣aj, for all 1 ≤ j ≤ n. Therefore d is a common divisor of a1,…,an.
  • Since d ∈ I, it is an R-linear combination of a1,…,an, i.e. d = a1b1 + + anbn for some b1,…,bn ∈ R. Now, if c is a common divisor of a1,…,an, then c divides any R-linear combination of a1,…,an, and so, in particular, c divides d.

Therefore, we have proved that d = gcd(a1,…,an) exists in R, and that d is an R-linear combination of a1 , … , an. Since we know (Proposition 1.26) that any two greatest common divisors of a given n-tuple are associated, we conclude that any gcd(a1,…,an) is an R-linear combination of a1,…,an too, and the theorem holds. __

A special case for consideration is when, given a,b ∈ R, we find gcd(a,b) ∈ R×. If R is a PID, then Bézout’s theorem shows that we can write 1 as an R-linear combination of a and b. (You may have seen this with R = ℤ.)

Definition 1.33. Let R be a commutative ring and a1,…,an ∈ R. We say that a1,…,an are coprime if their only common factors are the invertible elements of R.

For instance, let a,b ∈ ℤ. Then a and b are coprime if and only if a ∈ (ℤ∕b)×. Indeed, a and b are coprime if and only if there exists c,d ∈ ℤ such that 1 = ac + bd, which, modulo b, gives the equality 1 = ac ∈ ℤ∕b, and shows that a ∈ (ℤ∕b)×. Conversely, if a ∈ (ℤ∕b)×, then there exists u ∈ ℤ∕b such that au = 1 ∈ ℤ∕b, i.e. there exists v ∈ ℤ such that au + bv = 1 ∈ ℤ.

Corollary 1.34. Let R be a PID and a1,…,an ∈ R. Suppose that gcd(a1,…,an) ∈ R×. The following holds:

  1. 1 is an R-linear combination of a1,…,an.
  2. The ideal I = a1R + + anR generated by the ai’s is improper, i.e. I = R.

The assumption ‘R is a PID’ in Bézout’s cannot be weakened.

Example 1.35. Let R = ℤ[x]. Recall that R is an ID but not a PID. Let f = 2 and g = x, both regarded as polynomials in R. The only divisors of 2 in ℤ[x] are {±1,±2}, and those of x are {±1, ±x}. So their only common divisors are the invertible elements of ℤ[x], and we can take gcd(f,g) = 1. We observe that 1 is not a ℤ[x]-linear combination of 2 and x, indeed, suppose that there exist u,v ∈ ℤ[x] such that 1 = 2u + xv. Then, the constant term on the left hand side of the equality is odd, whereas that on the right hand side is even, a contradiction.

In other words, in ℤ[x] we could find a greatest common divisor of 2 and x, but we could not express it as a ℤ[x]-linear combination of 2 and x. (We’ll see in the next section the class of rings in which greatest common divisors always exist, and this class comprises ℤ[x].)
